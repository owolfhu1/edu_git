Git Cheat Sheet

Setup
  git init

Status + Log
  git status
  git diff
  git diff --staged
  git log

Staging + Commit
  git add <file>
  git add .
  git commit -m "message"

Branches
  git branch
  git branch -a
  git branch -d <name>
  git checkout -b <name>
  git checkout <name>
  git checkout -- <file>
  git switch -c <name>
  git switch <name>

Undo
  git restore <file>
  git restore --staged <file>
  git reset HEAD <file>
  git reset --hard <commit>

File Ops
  git rm <file>
  git mv <source> <dest>
