Welcome to the edu-git remote repo

This remote repo is here so you can explore how branches, commits, and
merge requests look once a repo is connected. Use the local terminal to
connect a repo and push your work, then compare branches and review
changes from the Remote Repo UI. You can also clone this repo to start
from a known remote history.

Quick start:
- Open the Remote Repo UI and select edu-git.
- Clone this repo with `git clone https://remote.mock/edu-git`.
- Or connect an existing local repo with `git remote add origin https://remote.mock/edu-git`.
- Push your branch with `git push -u origin main`.
